General architecture:

The Scanstore is built using the following technologies and frameworks:

·      Programming language server side Coldfusion Railo  http://www.getrailo.com/

·      Frameworks server side Coldspring ValidateThis

·      Frameworks clientside Sencha Ext js

All client server interaction goes via json remote proxies defined in coldspring.
